## 今すぐ動作確認する方法

### ステップ1: バックエンドを起動

ターミナルで以下を実行：

```bash
cd backend
source venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### ステップ2: ブラウザで確認

1. **APIドキュメントを開く**
   - ブラウザで http://localhost:8000/docs を開く
   - Swagger UIが表示されます

2. **銘柄情報を取得**
   - `/api/v1/stocks/{code}` をクリック
   - 「Try it out」→ `code` に `7203` を入力 → 「Execute」
   - トヨタ自動車の情報が表示されます

3. **株価データを取得**
   - `/api/v1/stocks/{code}/prices` をクリック
   - 「Try it out」→ `code` に `7203`、`period` に `1y` を入力 → 「Execute」
   - 株価データのリストが表示されます

### ステップ3: 直接URLで確認

ブラウザのアドレスバーに以下を入力：

- ヘルスチェック: http://localhost:8000/health
- 銘柄情報: http://localhost:8000/api/v1/stocks/7203
- 株価データ: http://localhost:8000/api/v1/stocks/7203/prices?period=1y

## 利用可能な銘柄コード（モックデータ）

- `7203` - トヨタ自動車
- `6758` - ソニーグループ
- `9984` - ソフトバンクグループ
