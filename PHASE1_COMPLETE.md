## 動作確認方法

詳細は `LAUNCH_CHECKLIST.md` を参照してください。

### クイックスタート

```bash
# 1. 環境変数ファイル作成
cd backend
# .envファイルを作成して USE_MOCK_PROVIDER=true を設定

# 2. データベース初期化
./scripts/init_db.sh

# 3. アプリケーション起動
./scripts/start_dev.sh
```
