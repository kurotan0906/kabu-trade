# バックエンド起動手順（Phase 1動作確認用）

## 現在の状況

- ✅ バックエンド仮想環境: 作成済み
- ✅ 主要な依存関係: インストール済み
- ✅ 環境変数ファイル: 作成済み（USE_MOCK_PROVIDER=true）
- ⚠️ Node.js: 未インストール（フロントエンドは後で対応）
- ⚠️ Docker: 未インストール（データベースは後で対応）

## バックエンドAPIのみで動作確認

データベースなしでも、モックデータでAPIの動作確認が可能です。

### 起動手順

```bash
# 1. バックエンドディレクトリに移動
cd backend

# 2. 仮想環境を有効化
source venv/bin/activate

# 3. バックエンドを起動
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 動作確認

1. **APIドキュメントにアクセス**
   - ブラウザで http://localhost:8000/docs を開く
   - Swagger UIが表示されることを確認

2. **ヘルスチェック**
   - ブラウザで http://localhost:8000/health を開く
   - `{"status": "healthy"}` が表示されることを確認

3. **銘柄情報取得APIを試す**
   - APIドキュメントで `/api/v1/stocks/{code}` を展開
   - `code` に `7203` を入力
   - 「Try it out」→「Execute」をクリック
   - レスポンスにトヨタ自動車の情報が返ることを確認

4. **株価データ取得APIを試す**
   - APIドキュメントで `/api/v1/stocks/{code}/prices` を展開
   - `code` に `7203`、`period` に `1y` を入力
   - 「Try it out」→「Execute」をクリック
   - レスポンスに株価データのリストが返ることを確認

### 利用可能な銘柄コード（モックデータ）

- `7203` - トヨタ自動車
- `6758` - ソニーグループ
- `9984` - ソフトバンクグループ

### 注意事項

- データベースが起動していない場合、一部の機能（データ保存）は動作しません
- モックデータはメモリ上で生成されるため、再起動するとデータはリセットされます
- Redisが起動していない場合、キャッシュ機能は動作しませんが、API自体は動作します
